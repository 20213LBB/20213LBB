## 🚀 Selected Projects

### ML-Based Soft Robot Surrogate Modeling and ROS2 Control Framework

An open-source framework integrating MATLAB simulation, PyTorch neural-network surrogate modeling, and ROS2 communication for soft robot control experiments.

**Highlights**

- PyTorch MLP surrogate model for nonlinear soft-robot dynamics
- ROS2 communication between MATLAB and Python
- Modular object-oriented software architecture
- Automated experiment execution and visualization

🔗 Repository:
https://github.com/20213LBB/soft-robot-mlp-surrogate
